jQuery Ribbon

About

This plugin generates a 3D ribbon effect for a given element.

Assumptions

  • The element should have a defined width.
  • The element should be positioned relative or absolute.
  • The element should have a background-colour declared to appreciate the effect.

Default Usage

$('{selector}').ribbon();

Attributes

darken_by: integer

This represents a percentage to darken the ribbon fold. The default is 20 (%).

ends: string

identify which 'ends' are to be added to the ribbon. Options: ['both','left','right'] The default is 'both'

triangle_width: integer

This defines the size of the triangle, in pixels. The default is 15.

direction: string

This describes which direction the ribbon should appear to be (with respect to the curled edges). The default is 'up'.
