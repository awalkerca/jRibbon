jRibbon

Create a 3D ribbon effect using jQuery.

jRibbon grew from a tutorial I found on creating a ribbon effect with CSS. I grew tired of repeating myself, so built a jQuery plugin to help reduce the amount of repeated CSS I would need to write.

Usage

$('{selector}').ribbon();

Attributes

Attribute Type Description Default
darken_by Integer This represents a percentage to darken the ribbon fold. (%) 20
ends String identify which 'ends' are to be added to the ribbon. Options: ['both','left','right'] 'both'
triangle_width Integer This defines the size of the triangle.(px) 15
direction String This describes which direction the ribbon should appear to be (with respect to the curled ends). Options: ['up','down'] 'up'

Assumptions

  • The element should have a defined width.
  • The element should be positioned relative or absolute.
  • The element should have a background-colour declared to appreciate the effect.
